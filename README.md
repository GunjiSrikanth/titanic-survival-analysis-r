# Titanic Survival Analysis using R

This project analyzes the Titanic dataset using R to understand passenger demographics and patterns.

## Tools Used
- R
- ggplot2
- Data Visualization
- Exploratory Data Analysis (EDA)

## Key Analysis
- Passenger distribution by gender
- Age distribution analysis
- Boxplot visualization of age
- Scatter plot analysis

## Visualizations
The project includes multiple graphs created using ggplot2.

- Data Analysis
- Data Visualization

- Statistical Insights
